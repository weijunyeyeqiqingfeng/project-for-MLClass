# 项目简介
这是一个利用CLIP和Stable Diffusion实现的免训练草图生成系统。通过修改提示词，我能控制生成草图的笔画复杂度、线条风格和艺术风格，不需要额外训练任何模型。

# 代码设计

- 三层控制机制：
    - 笔画复杂度：5个等级（从极简到高度详细）
    - 线条风格：细线、粗线、交叉线、点线
    - 艺术风格：草图、极简、卡通

- 评估体系：
    - CLIP相似度：衡量与“草图”概念的匹配程度
    - 边缘密度：计算草图的视觉复杂度
    - 笔画数量估计：近似统计独立笔画数量
    - 结构复杂度：分析频率分布

- 实验验证：
    - 笔画复杂度控制：验证复杂度等级与生成特征的相关性
    - 线条风格控制：验证不同线条风格的可区分性

- 关键发现：
    - 笔画复杂度等级与边缘密度呈强正相关（r ≈ 0.88）
    - 复杂度的提高让草图更接近“草图”概念（CLIP分数提升）
    - 不同线条风格在视觉特征和指标上差异明显
    - 模型对某些类别（如房屋）有强先验，生成效果更好

# 代码运行
- 修改configs.py中的参数：
    - TEST_PROMPTS：添加你想生成的物体
    - STROKE_CONFIG：调整复杂度等级对应的生成步数
    - LINE_STYLE_CONFIG：定义新的线条风格
